# assign_manage_system 人事管理系統

使用Vue+ElementUI結合完成的人事管理系統.

## 編譯方法

### 不進行編譯(development)
```sh
npm run dev
```
### 進行編譯(production)
```sh
npm run build
```

## 客製化
- 可透過更改Logo或者名稱進行客製化

## 功能介紹
- 可設定4種角色,老闆、會計、人事、主管、普通員工
- 員工可自行透過功能選單更改密碼及請假及基本操作
- 老闆擁有所有權限
- 會計擁有Double Check計算薪水最後關卡，薪水計算完成需經過會計最後確認
- 人事擁有管理職務、薪水、調整員工出席狀況、請假審核、計算薪水及重置員工密碼功能
- 主管可負責請假審核、出差核准
